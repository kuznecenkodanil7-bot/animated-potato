# RaidPunish for Fabric 1.21.11

Клиентский мод для Minecraft Fabric 1.21.11.

## Что делает

- При клике по нику игрока в чате открывает меню наказания.
- Даёт выбрать:
  - `/mute ник время причина`
  - `/warn ник время причина`
  - `/ban ник время причина`
  - `/ipban ник время причина`
- После выбора наказания открывает экран ввода срока:
  - `d` — дни
  - `h` — часы
- После этого открывает экран причины:
  - можно написать свою причину;
  - можно выбрать готовый шаблон по правилам 2.x / 3.x;
  - можно скопировать готовую команду;
  - можно сразу отправить её в чат.

## Как работает клик по нику

Мод пытается достать ник из стиля текста в чате (`insertion`, `click event`, `hover event`).
На серверах, где никнеймы в чате оформляются отдельным кликабельным/стилизованным компонентом, это обычно работает нормально.
Если конкретный чат-плагин рендерит сообщения нестандартно, может понадобиться доработка под формат сервера.

## Сборка

Требования:

- Java 21
- Gradle 8+

Сборка:

```bash
./gradlew build
```

Готовый jar появится в:

```bash
build/libs/
```

## Структура

- `client/` — точка входа клиента
- `mixin/` — перехват клика по чату
- `ui/` — экраны выбора наказания, срока и причины
- `model/` — типы наказаний и шаблоны правил
- `util/` — извлечение ника из текста чата

## Важно

Этот мод — клиентский помощник для модерации.
